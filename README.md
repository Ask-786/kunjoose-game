#### Simple Game

Simple game for 5th students in madrasa to learn past and future tenses

## Dependencies

- Node js
- Npm

### Installation

```bash

# Clone the repo

git clone https://github.com/Ask-786/kunjoose-game.git
cd kunjoose-game

# Run the following commands

chmod +x build.sh
./build.sh

```
